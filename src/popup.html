Instructions

If these steps don't work, check guidance from your local IT service.

  1. Go to mysignins.microsoft.com/security-info, sign in, and click Add sign-in method (if you don't see it on the page, click 'Security info').
  2. Select Microsoft Authenticator → Set up a different authentication app → Next → Can't scan the QR code? → copy the secret key.
  3. In Phone-Free 2FA, Add account → paste the secret key → codes appear.
  4. On your Microsoft account page, click Next → enter a code Phone-Free 2FA generates. Done.

Next time you log in, Microsoft may still default to the Authenticator app. Click "I can't use my Microsoft Authenticator app right now" → Use a verification code.

How does it work?

2-factor authentication (2FA) adds a security layer beyond your password. This makes digital life much more secure, because a hacker cannot get into your accounts with your password alone.

Phone-Free 2FA uses the most common approach to 2FA, namely time-based one-time passwords (TOTP). You get a secret key from e.g. your Microsoft account, pass it to Phone-Free 2FA, and it will generate a unique 6-digit code for you every 30 seconds. You can then copy this code and paste it into your (e.g.) Microsoft account to log in.

Phone-Free 2FA is safe, secure, and open-source. It runs completely locally on your device, and encrypts your secret keys with your master passphrase / touch ID. This never leaves your device.

Phone-Free 2FA is developed by computer scientists at the University of Oxford (Dr Ulrik Lyngs) and University of Maastricht (Dr Konrad Kollnig, Henry Tari), as part of their work with Centre for Digital Habits. The code is available at https://github.com/ulyngs/redd-phone-free-2fa.

Settings

You have two options:

1. Tap the ⋮ button for any of your accounts in Phone-Free 2FA, choose Edit, and copy the secret key into your new app.

2. Use Export backup above and choose Plain text URIs to create a file you can import into most authenticator apps.

Export Backup

A backup lets you recover your Phone-Free 2FA accounts if you e.g. reinstall your browser, or get a new computer. Keep your backup file safe & secure! To import from a backup, go to Settings.

Choose a password to protect the backup file.

⚠️ This will export all secrets as unencrypted plain text. Anyone with this file can access your accounts. Only use this for migrating to another authenticator app.

Import Backup

Restore accounts from a backup file. Supports backups from Phone-Free 2FA, as well as plain otpauth:// URI text file exports from other authenticator apps.

Enable Touch ID?

Unlock Phone-Free 2FA with your fingerprint instead of typing your passphrase each time.

When prompted, choose Google Password Manager (or another password manager like 1Password).

Do not select "Windows Hello", as this currently does not support secure processing of your fingerprint from a browser extension.
