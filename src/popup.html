2-factor authentication (2FA) adds a security layer beyond your password. This makes digital life much more secure, because a hacker cannot get into your accounts with your password alone.

The most common approach to 2FA is time-based one-time passwords (TOTP). You get a secret key from e.g. your university Microsoft account, and pass it to the 2FA service (e.g., this extension). The 2FA service then generates a unique 6-digit code for you every 30 seconds. ReDD 2FA encrypts your secret keys with your master passphrase / touch ID, which never leaves your device.

ReDD 2FA is safe, secure, and open-source. It is developed by computer scientists at the University of Oxford (Dr Ulrik Lyngs) and University of Maastricht (Dr Konrad Kollnig, Henry Tari), as part of the Reduce Digital Distraction Project. The code is available at https://github.com/ulyngs/redd-2fa.

Add Account

Services usually show the secret key as a QR code — but a QR code is just a fancy way of displaying it. There's always an option to reveal the key as text instead.

For university Microsoft accounts:

  1. Go to mysignins.microsoft.com/security-info and click Add sign-in method
  2. Select Microsoft Authenticator → Set up a different authentication app → Next → Can't scan the QR code? → copy the secret key
  3. Paste the secret key here in ReDD 2FA, then copy one of the generated codes and give it to Microsoft — done!

Note: When you next log in to your Microsoft acccount, it may still default to the Authenticator app for 2FA — click "Sign in another way"/"I can't use my Microsoft Authenticator app right now" → Use a verification code.

Delete Account

This action cannot be undone. Make sure you have a backup or alternative 2FA method.

Export Backup

Choose a password to protect the backup file.

⚠️ This will export all secrets as unencrypted plain text. Anyone with this file can access your accounts. Only use this for migrating to another authenticator app.

Import Backup

Restore accounts from a backup file. Supports backups from ReDD 2FA, as well as plain otpauth:// URI text file exports from other authenticator apps.

Enable Touch ID?

Unlock ReDD 2FA with your fingerprint instead of typing your passphrase each time.

Important for Windows users

When prompted, choose Google Password Manager (or another password manager like 1Password).

Do not select "Windows Hello" — it does not support the secure key derivation required for biometric unlock.
